Introduction

what is html?
HTML stands for hypertext markup Language used for structuring the web, it`s like the skeleton that form the basic structure of every web with css used for styling and javascript responsible for the dynamicity of the web

Basic structure of html includes:

  • HTML: the root document of html document
  • HEAD: contain the metadata about the document of the html, title and links
  • Title: sets the title of the page
  • BODY: contains the content of the html documentwhich is dispayed in the browser window
  • HEADINGS: Range from H1 to H6 in hierachy order with H1 being the largest and H6 being the smallest

Order and Unodered List

Ordered list: order list is defined in html with the OL tag and li tags nested inside it, ordered list tags automatically assign a number to each list items from 1 to the number of the items

UnOrdered list: Unorder list is defined in html with the UL tag and li tags nested inside it, Unordered list tags automatically assign a dots to each list items

  • The main difference between the list items is that order list assign number to each list items while unorder list items assign dots to each item
